RXN: Ask Lola


"Leading Online Learning Assistant, or LOLA for short, is a set of online tools available to subscribers of the Retail Exchange Network (RXN) Platform. It is a member only complimentary service offered by RXN."

HTML, CSS, Javascript


The goal of the Ask Lola was to improve the user experience and increase the efficiency of finding information about RXN and ORBIT.


In this project I worked with Retail Exchange Network to create a customer support website for the web application ORBIT. My role was to create and design a functional website that provides information and content that will assist users with navigating and operating ORBIT.


Along with this project, I given the task to create online video manuals using Photoshop and Powtoons.

Features:

  • Responsive
  • Online Video Manuals
  • Contact Forms
  • 30+ pages
  • Accordion sections
