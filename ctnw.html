Cards That Never Were


Cards That Never Were is a sports memorabilia website that connects with baseball enthusiasts and allows them to share their creativity with others.

XHTML, CSS, MySQL, Amazon Web Services


In this project I worked with Jack Opkins to create a sports memorabilia e-commerce website. I led a team of four to design, implement, test, and deploy a data-intensive web application. I was responsible for the design and front-end implementation of the website using JavaEE7 and Photoshop.


Features:

  • Responsive
  • PayPal shopping cart
  • Product Slider
  • Add/Remove Product function for client
  • Create blog posts for client
