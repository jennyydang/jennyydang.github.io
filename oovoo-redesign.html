ooVooo Redesign


ooVoo is a global social media platform for connecting with friends and family via high quality video chat, video messaging and rich, collaborative video stories.

Wordpress

In this project, I was given an assignment to find a website to redesign. I wanted to create a minimal yet fun website. The goal of this ooVoo Redesign was for me to take an existing parent theme, Tora, and create a child-theme to enhance the parent theme.


Features:

  • Responsive
  • Parallax scrolling
  • 404 page
  • CSS animations
  • Scroll-to-Top
  • Contact form plugin
