Jenny Dang - NYC Demographic Visualization

NYC Demographic Visualization

Data Visualization

"A visualization for users to identify the populations's ethnicity, gender, and use of public assistance."

Project Information

The dataset contains demographic statistics that is broken down by zip code in NYC. The goal of the NYC Demographic Visualization was to communicate information clearly through graphical means. I wanted the visualization to be simple, easy for the user to interact, and for them to facilitate instant understanding.

For this project, I led a team of four to design and develop an info visualization of NYC Demographic. We transferred CSV file to FusionTables in order to populate information to Google Maps. We also designed user-interface to be accessible and easy for users to obtain information quickly using HTML, CSS, Bootstrap, and Javascript.

Features

  • Filter Controls
  • Google Maps Search Engine
  • Reset Map Function
