ooVoo

Redesign + Wordpress

"ooVoo is one the world’s largest video and messaging apps. With over 100 million users from 130 countries and 54 MM in the US alone, we consider this app to be popular with kids under 25."

Project Information

In this project, I was given an assignment to find a website to redesign. I wanted to create a minimal yet fun website. The goal of this ooVoo Redesign was for me to take an existing parent theme, Tora, and create a child-theme to enhance the parent theme.

Features

  • Responsive Design
  • Parallax Scrolling
  • 404 page
  • CSS animations
  • Scroll-to-Top
  • Contact form plugin
