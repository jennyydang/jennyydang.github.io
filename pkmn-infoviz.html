Pokemon Stats Visualization

Data Visualization

"Pokemon Stats Visualization is an info visualization that contains the stats of the first generation of Pokemon (151 entries)."

Project Information

The dataset contains the data of the first generation Pokemons' Attack, Defense, Speed, Special Attack, and Special Defense. The goal of the Pokemon Stats Visualization was to communicate information clearly through graphical means. I wanted the visualization to be straight-forward, and easy to for the user to identify the strengths and weaknesses of each individual pokemon.

Features

  • Dynamic
  • Filter Types
  • Reset function
